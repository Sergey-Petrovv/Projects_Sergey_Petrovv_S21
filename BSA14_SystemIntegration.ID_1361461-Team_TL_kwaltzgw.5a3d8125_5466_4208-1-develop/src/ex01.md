# Exercise 01 — Определение мастер-систем

Для задачи 2 определите мастер-системы для входящих данных. 

1. В таблицу, созданную в ex.00, для каждого входящего потока данных добавьте возможные варианты систем в окружении, пригодные для получения данных. 
2. Для каждого входящего потока выберите и укажите мастер-систему.
3. Кратко обоснуйте выбор мастер-системы (укажите цель получения данных).
4. Результаты представьте в табличном виде.
